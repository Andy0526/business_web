文件为 网贷之家中,针对各平台的最近点评.

每条点评格式为:
{
    'platName':点评平台名
    'evaluation' : 评价者态度(0-不推荐,1-一般,2-推荐)
    'reviewContent': 评价内容
    'label':评价标签
    'reviewUserName':用户名
    'reviewDate':评价时间
}


如下:
{
    "reviewContent": "平台不错，，都是月标，，投资2000赚了200多。因为注册送了145代金券，，然后春节期间领取到红包100。",
    "reviewDate": "2016-04-14 14:40:18",
    "label": [
        "还不错"
    ],
    "reviewUserName": "8883662846",
    "platName": "睿银财富",
    "evaluation": "2"
}

建议:
可以考虑在平台档案页面做个最新滚动评论
